## Chosen Open Source Software
**Git**

Git is a distributed version control system created by **Linus Torvalds** in **2005**. It is used to track changes in files, manage source code, and support collaboration among developers. Git is one of the most important open-source tools in modern software development and is widely used through platforms such as GitHub, GitLab, and Bitbucket.

Git is licensed under the **GNU General Public License (GPL v2)**.

---

## Project Overview
This project is based on an audit of **Git** as part of the Open Source Software course. The purpose of this project is to study Git as an open-source software tool and understand its:

- origin and purpose
- open-source philosophy
- license and freedoms
- impact on software development
- role in the global open-source ecosystem

In addition to the audit topic, this repository also includes **five Linux shell scripts** that demonstrate basic shell scripting concepts such as variables, conditional statements, file handling, arithmetic operations, and command execution.

## Shell Scripts Included

### 1. `file_counter.sh`
**Description:**  
This script counts the number of files in a directory. It helps in quickly checking how many files are present in a given location.

**Purpose:**  
Useful for directory analysis and basic file management tasks.

---

### 2. `backup.sh`
**Description:**  
This script creates a backup of a file or directory.

**Purpose:**  
Useful for making safe copies of important data before modification or deletion.

---

### 3. `system_info.sh`
**Description:**  
This script displays basic system information such as system name, kernel details, current user, and other useful environment details.

**Purpose:**  
Useful for understanding the Linux system environment.

---

### 4. `check_file.sh`
**Description:**  
This script checks whether a specific file exists or not.

**Purpose:**  
Useful for file validation in shell scripting and automation tasks.

---

### 5. `calculator.sh`
**Description:**  
This script performs simple addition of two numbers entered by the user.

**Purpose:**  
Useful for demonstrating arithmetic operations and user input handling in Bash.

## Requirements
To run this project successfully, you need:

- A **Linux system** or Linux terminal environment
- **Bash shell**
- Basic command-line knowledge
- Git installed on the system

### Check Bash version
```bash
bash --version
```

### Check Git version
```bash
git --version
```

---

## How to Run the Scripts

### Step 1: Open terminal
Navigate to the project folder.

```bash
cd Open-Source-Audit-Git
```

### Step 2: Move into the scripts folder
```bash
cd scripts
```

### Step 3: Give execute permission to all scripts
```bash
chmod +x *.sh
```

### Step 4: Run any script
Use the following format:

```bash
./script_name.sh
```

Example:

```bash
./system_info.sh
```
